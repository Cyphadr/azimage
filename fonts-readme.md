# 中式复古主题字体说明

## 已更换为复古宋体风格字体

当前**中式复古主题**使用**复古宋体**风格字体，以民国风格为主，营造浓厚的书卷气与传统韵味。

### 字体列表（按优先级）

1. **朱雀仿宋 Zhuque Fangsong** ⭐ 主力复古字体（需本地安装）
   - 特点：改刻自民国1932年"南宋"活字，中宫收紧，撇捺舒展
   - 风格：民国活字韵味 + 浓厚书卷气 + 含蓄秀美
   - 设计师：韩佑之（原稿）、邹根培（雕刻）、璇玑造字（改编）
   - 授权：OFL 1.1（完全免费商用）
   - 下载：https://github.com/TrionesType/zhuque

2. **思源宋体 Source Han Serif SC** - CDN 自动加载
   - 特点：Adobe 开源宋体，清晰易读
   - 风格：现代宋体，专业排版
   - 授权：OFL 1.1（完全免费商用）
   - 加载：jsDelivr CDN（自动）

3. **Noto Serif SC（思源宋体）** - Google Fonts 备选
   - 特点：Google Fonts 版本的思源宋体
   - 加载：Google Fonts CDN（自动）

4. **SimSun（宋体）** - 系统字体保底
   - 确保在任何情况下都能正常显示

### 优势

✅ **复古韵味**：朱雀仿宋完美还原民国活字风格，适合文学、古籍类排版  
✅ **灵活配置**：支持本地字体 + CDN 备选，双重保障  
✅ **免费可商用**：所有字体均为开源字体（OFL 1.1 协议）  
✅ **无需强制本地字体**：即使不安装朱雀仿宋，也会自动使用思源宋体 CDN

### 使用方法

#### 方案一：完整体验（推荐）

1. 下载朱雀仿宋字体：https://github.com/TrionesType/zhuque/releases
2. 将字体文件放入项目的 `fonts/` 目录（参考 `fonts-setup-guide.md`）
3. 在浏览器中打开 `preview.html`，点击"**中式复古**"主题按钮
4. 查看朱雀仿宋的复古效果

#### 方案二：快速预览（无需下载字体）

1. 直接在浏览器中打开 `preview.html`
2. 点击"**中式复古**"主题按钮
3. 系统会自动使用思源宋体 CDN（效果略有不同，但仍为宋体风格）

### 详细配置指南

请查看 **`fonts-setup-guide.md`** 获取完整的字体下载、安装和配置说明。

## 字体对比

### 朱雀仿宋 vs 思源宋体

| 特性 | 朱雀仿宋 | 思源宋体 |

| **风格** | 民国活字韵味，书卷气浓 | 现代宋体，清晰专业 |
| **笔画** | 粗细变化强烈，撇捺舒展 | 笔画均匀，适合屏幕阅读 |
| **中宫** | 收紧，重心靠右下 | 中正，结构规整 |
| **适合场景** | 文学作品、古籍复刻、艺术排版 | 正文阅读、专业出版 |

- **朱雀仿宋**：

  - ✅ 复古风格设计
  - ✅ 需要浓厚书卷气的场景
  - ✅ 艺术性展示

- **思源宋体**：
  - ✅ 正文阅读
  - ✅ 专业文档

## 技术说明

### 字体加载策略

1. **优先本地**：首先尝试加载 `fonts/` 目录中的朱雀仿宋
2. **系统安装**：检查操作系统是否安装了朱雀仿宋
3. **CDN 备选**：从 jsDelivr CDN 加载思源宋体
4. **Google Fonts**：从 Google Fonts 加载 Noto Serif SC
5. **系统字体**：最终使用系统自带的宋体

### 字体格式支持

- **WOFF2**：现代浏览器推荐，文件小、加载快
- **WOFF**：兼容旧版浏览器
- **TTF**：通用格式，所有浏览器支持
- **本地字体**：通过 `local()` 函数调用系统已安装的字体

### 性能优化

- 使用 `font-display: swap` 确保文字立即显示
- 分割加载思源宋体（按 unicode-range），减少初始加载大小
- 本地字体优先，减少网络请求

## 常见问题

### Q: 为什么推荐使用朱雀仿宋？

朱雀仿宋是专门为中文正文排版设计的开源仿宋字体，具有浓厚的民国活字韵味和书卷气，非常适合文学作品、小说等内容的排版。它的笔画粗细变化强烈，中宫收紧、撇捺舒展，能够营造出独特的复古氛围。

### Q: 不下载朱雀仿宋可以用吗？

完全可以！如果不下载朱雀仿宋，系统会自动使用思源宋体 CDN，同样是宋体风格，只是风格略有不同（思源宋体更现代化，朱雀仿宋更复古）。

### Q: 如何确认朱雀仿宋是否加载成功？

1. 打开 `preview.html`
2. 按 `F12` 打开浏览器开发者工具
3. 切换到"**Console**"标签
4. 在控制台输入：`getComputedStyle(document.querySelector('.az-cover__title')).fontFamily`
5. 如果看到 `"Zhuque Fangsong"` 或 `"朱雀仿宋"`，说明加载成功

### Q: 字体文件放在哪里？

在项目根目录创建 `fonts/` 文件夹，将字体文件放入：

```
az/
├── css/
├── fonts/                    ← 创建这个文件夹
│   ├── ZhuqueFangsong-Regular.ttf
│   └── ZhuqueFangsong-Regular.woff2
├── preview.html
└── ...
```

详细步骤请查看 `fonts-setup-guide.md`。

### Q: 思源宋体 CDN 加载慢怎么办？

如果 jsDelivr CDN 加载较慢，可以：
1. 下载思源宋体到本地 `fonts/` 目录
2. 修改 `css/az-theme-retro-fonts.css` 添加本地路径
3. 或者直接使用系统自带的宋体

### Q: 可以商用吗？

完全可以！朱雀仿宋和思源宋体都采用 OFL 1.1 开源许可证，支持免费商用，包括但不限于：
- ✅ 印刷出版
- ✅ LOGO 设计
- ✅ 网页设计
- ✅ 视频制作
- ✅ 软件嵌入
